在8110上运行Debian(带有GUI)
Created2021-03-10|Updated2021-03-10
|Post View:|Comments:

运行截图

首先，你需要一个基于Debian的环境（非GNU或Debian-Based没有经过测试，可能会导致一些问题）

已经root的8110

一张SD卡（第一分区FAT，第二分区Ext4，分区大小想必想安装的人都知道吧）分区后插进手机

构建：

克隆这些项目

之后安装pbuilder

设置Build chroot

最后以root身份运行make(先试运行，来确认没有恶意行为)

QEMU引导模式：

在QEMU用户仿真器中运行debootstrap（第二阶段）和其他各种设置命令。此模式需要qemu user static包。要启用它，请在make或make install To device命令后面附加USE_QEMU=1：

之后安装开始

WLAN设置：

在构建过程中，会提示你修改一个“/etc/wpa_supplicant.conf”文件

SSH：

安装完成后，通过路由器找到手机IP（假设X.X.X.X）

root权限，请用sudo

窗口列表：

显示打开窗口的列表，在滑盖开启的情况下，按电源键。

Bug们：

colorgrid应用程序并不总是工作（它已从主屏幕中删除，但可以通过ssh作为colorgrid启动）

注意！

本项目仍在测试，风险将由用户自行承担。
